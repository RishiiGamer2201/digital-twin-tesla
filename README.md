# Digital Twin of Nikola Tesla

**AIMS DTU Summer Project 2026**

A Digital Twin of Nikola Tesla, grounded in his actual writings, patents, and interviews. Ask Tesla about his inventions, philosophy, and vision for the future. Powered by RAG (Retrieval-Augmented Generation), long-term memory, and Google Gemini AI.

---

## Features Overview

### Core AI Features

#### 1. RAG Pipeline (Retrieval-Augmented Generation)
Every response Tesla gives is grounded in real source material. The system retrieves relevant passages from Tesla's autobiography ("My Inventions"), Wikipedia articles, Wikiquote collections, and YouTube documentary transcripts before generating a response. This means Tesla's answers are not hallucinated but are backed by actual historical text. When Tesla says "I discovered the rotating magnetic field while walking in Budapest," that claim is pulled directly from his autobiography.

#### 2. Deep Persona Engine
Tesla's personality is encoded in a 3000+ token system prompt covering his speech patterns (formal Victorian English with Serbian touches), epistemic style (favoring mental visualization over brute-force experimentation), teaching approach (vivid analogies to nature), and timeline awareness (knowledge cutoff at January 1943). The persona includes his opinions on Edison, his friendship with Mark Twain, his love for pigeons, and his passion for free wireless energy. Every response sounds authentically like Tesla.

#### 3. Conversation Memory (Short-Term + Long-Term)
The system remembers conversations across two tiers:
- **Short-term**: A rolling buffer of the last 20 messages provides immediate context within a session. Tesla remembers what you just said.
- **Long-term**: When you end a session, Gemini generates a structured summary (topics discussed, key facts, entities mentioned) and stores it in a SQLite database. In future sessions, relevant past memories are retrieved by keyword matching and injected into Tesla's context. Tell Tesla your name once, and he remembers it in your next session.

#### 4. Source Citations
Every response shows which Tesla writings were referenced, displayed as clickable chips below the message. Click a chip to expand a preview of the actual source text. This lets users verify that Tesla's claims are historically grounded and trace responses back to specific documents.

#### 5. Fact Confidence Badges
Each response is tagged with a confidence level based on RAG retrieval scores:
- **"From my writings"** (green) - Response is strongly grounded in Tesla's actual texts (high similarity score)
- **"From my recollection"** (amber) - Response is partially grounded, with some persona inference
- **"Beyond my time"** (red) - Response involves speculation or topics after 1943

This helps users distinguish between historically verified claims and AI-generated extrapolations.

---

### Interactive Features

#### 6. Streaming Responses with Typing Animation
Responses stream token-by-token from the Gemini API via Server-Sent Events (SSE). Instead of waiting 5-10 seconds for a full response, text appears character-by-character in real time, creating a natural typing effect. This makes the app feel alive and responsive.

**Technical Details**: The backend uses Flask's `Response` generator with `text/event-stream` MIME type. The frontend reads the stream using the Fetch API's `ReadableStream` reader and parses SSE events incrementally.

#### 7. Topic Suggestion Chips
After each response, 2-3 clickable follow-up question chips appear below the message. These are generated by Gemini based on the current conversation context. For example, after discussing the Tesla Coil, suggestions might include "How does resonance work in the Tesla Coil?", "What was your Colorado Springs experiment?", and "Did you ever achieve wireless power transmission?". Clicking a chip automatically sends that question.

#### 8. Emotion Indicator
A colored dot on Tesla's avatar changes based on the emotional tone of his response:
- **Gold** (excited) - When discussing his inventions or breakthroughs
- **Red** (bitter) - When discussing Edison, business failures, or exploitation
- **Purple** (nostalgic) - When recalling his childhood, Serbia, or old friends
- **Blue** (philosophical) - When discussing the nature of energy, the universe, or humanity's future
- **Cyan** (neutral) - Default conversational tone

The emotion is detected by a separate Gemini call that classifies the response's tone.

#### 9. Interactive Timeline (1856-1943)
A visual vertical timeline showing 16 key events in Tesla's life, from his birth during a lightning storm in Smiljan (1856) to his death at the New Yorker Hotel (1943). Each event has a year, title, and description. The timeline features:
- Glowing connecting line with animated dots
- Hover effects on event cards
- Click any event to automatically ask Tesla about it in the chat

Events include: Born in Smiljan, Graz Polytechnic, Budapest Revelation, Arrived in America, Left Edison, Tesla Electric Company, Westinghouse Partnership, Tesla Coil, Chicago World's Fair, Niagara Falls, Colorado Springs, Wardenclyffe Tower, Wardenclyffe Abandoned, AIEE Edison Medal, Death Ray Claims, and Passed Away.

#### 10. Knowledge Graph Visualization
An interactive canvas-based graph showing the connections between Tesla's inventions, the people in his life, key places, and core concepts. The graph contains ~18 nodes:
- **People** (gold): Tesla, Edison, Westinghouse, Mark Twain, J.P. Morgan
- **Inventions** (cyan): AC Power, DC Power, Tesla Coil, Radio, X-rays
- **Places** (purple): Wardenclyffe, Colorado Springs, Niagara Falls, Serbia
- **Concepts** (green): Rotating Field, Wireless Energy, War of Currents, Patents

Nodes are connected by edges showing relationships. A simple force-directed layout positions nodes automatically. Click any node to ask Tesla about that topic in the chat.

#### 11. Quiz Mode
An interactive multiple-choice quiz about Tesla and electrical engineering. Features:
- Questions generated by Gemini on demand, so they are never repeated
- 4 options per question (A, B, C, D)
- Visual feedback: correct answers highlight green, wrong answers highlight red
- Tesla grades your answer in his own voice (e.g., "Excellent! You understand the principle of resonance.")
- Running score tracker (correct / total)
- "Next Question" button for unlimited quizzes
- Optional topic filter for focused study

#### 12. Tesla vs Edison Debate Mode
A split-panel debate interface where Tesla and Edison argue their positions on any topic. Features:
- Enter any debate topic (e.g., "AC vs DC power", "The role of theory in invention", "Should energy be free?")
- Tesla's response appears on the left (cyan accent), Edison's on the right (amber accent)
- Both responses are generated by Gemini using separate persona prompts
- Edison's persona is historically accurate: practical, business-minded, favoring brute-force experimentation, skeptical of Tesla's grand visions
- Great for understanding both sides of historical technical debates

#### 13. PDF Upload
Users can upload their own Tesla-related PDFs (research papers, patent documents, additional writings) to expand the knowledge base on the fly. The system:
- Extracts text from the PDF using pypdf
- Chunks the text into 512-word segments with 64-word overlap
- Generates embeddings using the local sentence-transformer model
- Adds the chunks to the ChromaDB vector store
- Reports how many chunks were added
- Future chat responses will reference the uploaded material

### UI/UX Features

#### 14. Dark/Light Theme Toggle
A sun/moon toggle button in the header switches between dark mode (default, electric blue on deep navy) and light mode (blue accents on clean white). The theme preference is saved in localStorage and persists across browser sessions. All colors transition smoothly via CSS custom properties.

#### 15. Mobile Responsive Design with Sidebar Drawer
On screens smaller than 768px, the sidebar collapses and a hamburger menu button appears. Tapping it slides the sidebar in from the left as an overlay with a backdrop. Tapping the backdrop closes it. All features remain fully accessible on mobile.

#### 16. Voice Input and Output
- **Voice Output (TTS)**: Uses the browser's built-in Speech Synthesis API. Tesla's responses are read aloud with a male English voice, rate 0.9, pitch 0.85. Toggle on/off in the sidebar.
- **Voice Input (Mic)**: Uses the browser's Web Speech API for speech recognition. Click the microphone button, speak your question, and it is automatically transcribed and sent. Works in Chrome and Edge.
- No server-side dependencies for voice. Everything runs in the browser.

#### 17. Conversation Export
Download the entire chat as a formatted Markdown (.md) file. The export includes:
- Title: "Conversation with Nikola Tesla"
- Export timestamp
- All messages with role labels (You / Nikola Tesla)
- Clean formatting with separators between messages

Click "Export Chat" in the sidebar to download.

#### 18. Memory Dashboard
The sidebar shows live statistics:
- Total sessions stored in long-term memory
- Messages in the current session
- Past session summaries with delete buttons
- Each past session can be individually deleted

#### 19. Session Management
- **Save Session**: Ends the current session, generates a Gemini-powered summary, and stores it in SQLite for future recall
- **Clear Chat**: Wipes the current conversation without saving to long-term memory
- **Delete Past Session**: Remove individual sessions from long-term memory

## Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/RishiiGamer2201/digital-twin-tesla.git
cd digital-twin-tesla
```

### 2. Create virtual environment and install dependencies

```bash
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # macOS/Linux
pip install -r requirements.txt
```

### 3. Set up environment variables

```bash
copy .env.example .env       # Windows
# cp .env.example .env       # macOS/Linux
```

Edit `.env` and add your free Gemini API key from [aistudio.google.com](https://aistudio.google.com):

```
GEMINI_API_KEY=your_api_key_here
SCIENTIST_NAME=Nikola Tesla
```

### 4. Collect Tesla's writings (one-time)

```bash
python scripts/collect_data.py
```

Downloads Tesla's autobiography, Wikipedia data, Wikiquote, and YouTube transcripts.

### 5. Build the vector store (one-time)

```bash
python scripts/build_vectorstore.py
```

Chunks and indexes 188 documents into ChromaDB using local embeddings.

### 6. Run the application

```bash
python server.py
```

Open **http://localhost:5000** in your browser.
